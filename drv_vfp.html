Czech Exchange Format for Land Consolidation

(Starting with GDAL 2.1.0)

VFP (the Czech Exchange Format for Land Consolidation) is a light-weight XML data format for the interchange of Land Consolidation data between applications in the Czech Republic.

OGR has support for VFP reading if GDAL is build with expat library support.
